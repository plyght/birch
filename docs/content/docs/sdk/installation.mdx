## npm

```bash
npm install @inaplight/birch-client
```

## Bun

```bash
bun add @inaplight/birch-client
```

## yarn

```bash
yarn add @inaplight/birch-client
```

## pnpm

```bash
pnpm add @inaplight/birch-client
```

## Package Information

- **Package name**: `@inaplight/birch-client`
- **Version**: Check [npm](https://www.npmjs.com/package/@inaplight/birch-client) for latest version
- **TypeScript**: Full TypeScript support included
- **Node.js**: Requires Node.js 18 or higher

## Verify Installation

After installation, verify the package is installed correctly:

```bash
npm list @inaplight/birch-client
```

You should see the package listed with its version.

## TypeScript Support

The SDK includes full TypeScript definitions. No additional `@types` package needed.

```typescript
import { configureBirch, daemonClient } from '@inaplight/birch-client';

// All types are available
const result: RotateResult = await daemonClient.rotate('MY_KEY');
```
